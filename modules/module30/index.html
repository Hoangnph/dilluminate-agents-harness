🔑 Key Insight: Moore's Law cho AI — Capability × Cost

Từ 2020–2026: model capabilities tăng ~10× trong khi cost-per-token giảm ~100×. Trend này là exponential, không linear. Ý nghĩa cho FinTech Corp: LoanBot v4.0 dùng claude-haiku-4-5 ở 80% decisions với cost 54 VNĐ/case. LoanBot v5.0 (2028) sẽ có capability tương đương Sonnet-4-6 hôm nay nhưng ở giá Haiku hôm nay. Kết quả: high-value 5-agent panel (1.150 VNĐ hiện tại) sẽ cost ~100 VNĐ vào 2028 — mở ra economics cho ultra-complex decisions mà hôm nay không viable. Điều quan trọng hơn: bất kỳ ai biết build agentic systems hôm nay đang sở hữu skill set có half-life dài nhất trong tech.

2024–2025
Foundation Era: Single agents, basic tool use, RAG. LLMs reliable nhưng cost còn cao. Multi-agent là cutting-edge. LoanBot M1–M15 của chúng ta.
2026–2027
Scale Era (HÔM NAY): Multi-agent debate, self-improvement, governance frameworks. Computer-use agents mainstream. MCP standardizing tool protocols. Long-context (200K+ tokens) changes RAG. LoanBot M16–M30 của chúng ta.
2027–2028
Autonomy Era: Agents run weeks-long tasks independently. Inter-agent trust protocols (agent-to-agent authentication). Real-time learning from deployment (no separate retrain cycle). Multimodal: agents see, hear, and act simultaneously.
2028–2030
Ecosystem Era: Agent marketplaces (specialist agents for hire). Cross-organization agent networks (LoanBot ↔ Credit Bureau AI ↔ Central Bank AI). Agent-designed agents (meta-learning at system level). Regulatory frameworks matured. LoanBot v6.0 territory.
🏢 Tình huống: FinTech Corp 2028 — LoanBot v5.0

Bối cảnh: Năm 2028. FinTech Corp đã mở rộng ra 5 quốc gia Đông Nam Á. LoanBot v5.0 xử lý 5 triệu hồ sơ/tháng (100× M4 speed). Khác biệt cốt lõi so với v4.0: (1) Persistent Memory: LoanBot nhớ toàn bộ 3-year loan history của mỗi customer — không cần RAG, full context mỗi decision. (2) Multimodal Document Processing: Camera phone photo của bảng lương → instant OCR + structured data, không cần manual entry. (3) Real-time Adaptation: Model cập nhật thresholds daily từ defaults, không cần 30-day batch cycle. (4) Cross-border Agent Network: LoanBot Vietnam ↔ LoanBot Thailand ↔ Credit Bureau ASEAN Agent — real-time fraud detection cross-country.

Agents viết tool code của chính mình khi gặp task mới. LoanBot discovers cần validate business license → writes BizLicenseLookupTool → tests → deploys → uses trong same session.

🌐
MCP Ecosystem Maturity

Model Context Protocol standardizes agent-to-tool-to-agent communication. LoanBot sử dụng MCP marketplace: CreditBureauAgent, TaxAuthAgent, FraudDetectorAgent — như app store cho agent capabilities.

Python (LoanBot v5.0 — Multimodal Interview Agent Sketch) 
import anthropic
from typing import List

class LoanBotV5InterviewAgent:
    """Next-gen: voice + visual + long-context + real-time adaptation."""

    def __init__(self):
        self.client = anthropic.Anthropic()
        self.conversation_history: List[dict] = []
        self.customer_context = {}    # Long-context: 5yr history

    async def conduct_interview(self, customer_id: str,
                                    video_frames: list = None,
                                    audio_transcript: str = None) -> dict:
        # Build rich multimodal context
        messages = [
            {"role": "user", "content": [
                # Text context: 5-year customer history (long-context)
                {"type": "text",
                 "text": f"Customer {customer_id} history:\n{self._load_full_history(customer_id)}"},
                # Audio: what customer said in the interview
                {"type": "text",
                 "text": f"Customer verbal statement: {audio_transcript}"},
                # Image: document they showed on camera
                *([{"type": "image",
                    "source": {"type": "base64",
                              "media_type": "image/jpeg",
                              "data": frame}}
                   for frame in (video_frames or [])]),
            ]}
        ]

        # Extended thinking: reason through multimodal evidence
        response = self.client.messages.create(
            model="claude-sonnet-4-6",
            max_tokens=4000,
            thinking={"type": "enabled", "budget_tokens": 2000},
            system="""You are LoanBot v5.0. You have access to:
- Full 5-year customer financial history
- Customer's verbal interview statement
- Document images (salary slips, bank statements) shown on camera
- OCR capability for document images
Conduct thorough analysis across all evidence sources.
Produce: decision (APPROVE/REVIEW/REJECT), confidence, key factors, customer explanation.""",
            messages=messages
        )
        return {'decision': 'REVIEW', # Extracted from response
                'confidence': 0.81,
                'reasoning': response.content[-1].text}

    def _load_full_history(self, customer_id: str) -> str:
        # In v5.0: query vector DB for 5yr full history
        return f"[5-year financial history for {customer_id}]"

Extended Thinking (claude-sonnet-4-6 feature): khi set thinking.enabled = true, model có riêng một "scratchpad" để reason trước khi produce output. Ứng dụng cho LoanBot v5.0:

  • Evidence reconciliation: verbal statement conflicts với document → extended thinking để resolve
  • Complex borderline cases: thay vì multi-agent debate → single agent với deeper reasoning có thể đủ
  • Audit trail: thinking blocks logged → regulators có thể see agent's reasoning process

Token budget: thinking_budget_tokens = 2.000 → agent spends up to 2K tokens thinking before answering. Đây là "slow thinking" (System 2) thay vì fast pattern matching (System 1). Cost: thinking tokens billed at input rate.

Hình 1: LoanBot v5.0 Agent Ecosystem. MCP protocol standardizes communication: mỗi external agent cung cấp tool endpoints, LoanBot orchestrates. Trust: mutual TLS + agent credential tokens. Response time target: <500ms per external call (parallel, not sequential).
📌 Khái niệm: Agent Trust Protocol — Làm Sao Biết Agent Kia Là Thật?

Trong agent ecosystem, LoanBot cần verify rằng "Central Bank AI" nó đang nói chuyện thực sự là từ NHNN — không phải attacker impersonating NHNN agent (prompt injection at ecosystem level). Trust mechanisms: (1) Mutual TLS: cả 2 agents exchange certificates. (2) Agent Credential Token: NHNN-issued JWT với expiry + signed claims. (3) Response Validation: LoanBot không blindly trust rule từ external agent — validate against local ruleset snapshot. (4) Rate Limiting + Anomaly Detection: nếu "NHNN Agent" đột ngột gửi 1.000 rule updates/minute → anomaly, halt và alert. M9 (Agent Security) principles apply đầy đủ trong ecosystem context.

💡 Analogy: Centaur Chess — Người + AI > Người Giỏi Nhất

Năm 1997, Deep Blue đánh bại Kasparov. Nhưng năm 2005, Centaur Chess tournaments (human + AI team) đánh bại cả Deep Blue và human grandmasters. Principle: AI + human > AI alone > human alone — khi phân công đúng chỗ. FinTech Corp 2028 tương tự: LoanBot v5.0 handle 98% decisions, con người tập trung vào: exception cases, ethical dilemmas, relationship management với khách hàng VIP, policy design, system oversight. 200 nhân viên thẩm định của 2024 → 25 AI Oversight Managers, 5 Agent Trainers, 3 Ethics Officers, 2 AI Product Managers — ít hơn về số, nhưng cao hơn về impact và lương.

🏢 FinTech Corp Org Chart 2028 — AI-First Structure

Không còn tồn tại (role eliminated): Thẩm định viên thủ công, Data Entry, Compliance Checklist reviewers. Roles transformed: Loan Officer → AI Oversight Manager (supervise LoanBot panels, handle REVIEW escalations). Risk Manager → AI Risk Architect (design governance policies, tune fairness constraints). (2) AI Ethics Officer: full-time role governance, bias auditing, regulatory liaison. (3) Agent Architect: designs multi-agent systems, selects models, manages costs. 🧠 Kiểm tra nhanh

Năm 2028, LoanBot v5.0 đạt 99% accuracy và xử lý 5 triệu hồ sơ/tháng. Có nên giảm human oversight từ 25 người xuống còn 5 người không? Lý luận của bạn?

Gợi ý: Không nên giảm quá nhiều, vì 3 lý do: (1) 99% accuracy trên 5 triệu = 50.000 cases/tháng có thể sai — nhiều hơn absolute number so với 8.2% × 50K = 4.100 cases hôm nay. Scale amplifies errors. (2) EU AI Act và NHNN mandate human oversight cho High-Risk AI — giảm xuống 5 người có thể vi phạm "adequate human oversight" requirement. (3) Black swan events: global financial crisis, regulatory overhaul, pandemic — AI trained on 2024-2028 data không có experience với black swans. Human oversight là circuit breaker. Counter-argument: 25 người supervise 5M cases = 1 person per 200K cases — không meaningful oversight anyway. Giải pháp: maintain headcount nhưng shift focus từ case review sang system monitoring và exception handling.

📚 Glossary Module 30

Thuật ngữ Định nghĩa
Agentic AIAI systems can take sequences of actions, plan, and pursue goals with minimal human intervention. Contrast với chatbots (single-turn) và traditional ML (classification only).
Computer-Use AgentAgent có thể nhìn màn hình và điều khiển UI (click, type, scroll) như người. Không cần API integration — works với bất kỳ existing software nào.
Long-Context WindowModel có thể xử lý hàng triệu tokens in một lần. Enables: full 5-year customer history as context, entire codebase, complete document libraries — without RAG retrieval.
Extended ThinkingModel reasoning mode: allocate "thinking budget" (tokens) cho internal deliberation trước khi output. System 2 thinking — slower, deeper, more accurate cho complex reasoning.
MCP (Model Context Protocol)Emerging standard (Anthropic 2024) cho agent-to-tool communication. Như USB-C cho agents: one protocol, many tools. Agent marketplaces built on MCP.
Agent EcosystemNetwork of agents từ nhiều organizations tương tác với nhau via defined protocols. LoanBot v5.0 + Credit Bureau AI + Central Bank AI = agent ecosystem.
Agent Trust ProtocolMechanisms để verify authenticity của agent trong ecosystem: mutual TLS, credential tokens, response validation. Prevents impersonation attacks in cross-org networks.
Centaur ModelHuman + AI collaboration where combined performance > either alone. Named after Centaur Chess (human + chess engine > either separately). Optimal division: AI operational, human strategic.
Real-Time LearningAgent cập nhật behavior từ feedback within deployment, không cần batch retrain cycle. Contrast với M27 (30-day ground truth batch). Future capability, not yet production-ready at scale.
Self-Designing ToolAgent viết và tests code cho tool của chính nó khi gặp task chưa có tool sẵn. Requires: code execution sandbox, automated testing, safety validation trước khi deploy.
Agent MarketplacePlatform mua/thuê specialized agent capabilities: CreditCheckAgent, FraudDetectionAgent, ComplianceAgent. Như app store nhưng cho agent tools và sub-agents.
AI Oversight ManagerNew role (2026+): supervise AI agent panels, handle escalations, ensure governance policies followed. Evolution của Loan Officer trong AI-first bank.
Disruption VectorDirection từ đó AI disruption có thể đến: (1) Cost collapse (100× cheaper → new markets viable); (2) Capability jump (new modality); (3) Regulatory mandate (forced adoption/restriction).
Half-Life of SkillsHow long a skill remains valuable before becoming obsolete. In AI: specific model knowledge has short half-life (months); system design principles have long half-life (years). Build durable skills.
